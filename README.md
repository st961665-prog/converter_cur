# 💱 Simple Currency Converter

Простой и удобный конвертер валют, разработанный на Python с использованием фреймворка **Streamlit**. Приложение позволяет получать актуальные курсы валют в реальном времени, используя открытый API [Frankfurter](https://www.frankfurter.app/).

## 🚀 Функциональные возможности

-   **Актуальные данные**: Получение курсов валют в реальном времени.
-   **Поддержка множества валют**: Выбор из широкого списка поддерживаемых мировых валют.
-   **Двусторонняя конвертация**: Отображение прямого и обратного курса.
-   **Кэширование**: Оптимизация загрузки списка валют для ускорения работы приложения.
-   **Интуитивный интерфейс**: Простой и понятный UI, созданный с помощью Streamlit.

## 🛠 Технологии

-   **Python 3.x**
-   **Streamlit**: Для создания веб-интерфейса.
-   **Requests**: Для выполнения HTTP-запросов к API.
-   **Frankfurter API**: Бесплатный источник данных о курсах валют (не требует API ключа для базового использования).
